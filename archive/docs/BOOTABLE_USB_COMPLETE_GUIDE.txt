## 📥 STEP 1: DOWNLOAD REQUIRED FILES

### A. Download Ubuntu ISO

1. **Go to:** https://ubuntu.com/download/desktop
2. **Click:** Download Ubuntu 24.04 LTS
3. **File:** ubuntu-24.04-desktop-amd64.iso
4. **Size:** ~3GB (takes 5-15 minutes depending on connection)
5. **Save to:** Your Downloads folder

### B. Download MJR-EHR (Already Done!)

You already have from Claude:
- ✅ mjr-ehr-system-LIGHTWEIGHT.tar.gz (27MB)

### C. Download USB Flashing Tool

**Windows - Rufus:**
1. Go to: https://rufus.ie
2. Download: rufus-4.x.exe (portable version)
3. No installation needed

**Mac/Linux - Etcher:**
1. Go to: https://www.balena.io/etcher
2. Download for your OS
3. Install the application

---

## 💾 STEP 2: CREATE BOOTABLE UBUNTU USB

### For Windows (Using Rufus):

1. **Insert USB drive** (8GB+)
   - WARNING: All data will be erased!

2. **Run Rufus** (rufus.exe)

3. **Configure Rufus:**
   - Device: Select your USB drive
   - Boot selection: Click "SELECT" → Choose ubuntu-24.04-desktop-amd64.iso
   - Partition scheme: GPT
   - Target system: UEFI (non CSM)
   - Volume label: UBUNTU_DEMO
   - File system: FAT32

4. **Click START**

5. **If prompted:** "Write in ISO Image mode" → Click OK

6. **Wait 5-10 minutes** for flashing to complete

7. **Click CLOSE** when done

8. **DO NOT EJECT YET!** - Ubuntu USB will re-mount

### For Mac/Linux (Using Etcher):

1. **Insert USB drive** (8GB+)

2. **Open Etcher**

3. **Flash from file:**
   - Click "Flash from file"
   - Select ubuntu-24.04-desktop-amd64.iso

4. **Select target:**
   - Click "Select target"
   - Choose your USB drive

5. **Flash!**
   - Click "Flash!"
   - Enter password if prompted

6. **Wait 5-10 minutes**

7. **USB re-mounts automatically**

---

## 📦 STEP 3: ADD MJR-EHR TO BOOTABLE USB

After flashing, the USB re-mounts as "UBUNTU_DEMO" or similar.

### A. Extract MJR-EHR Archive

**Windows (7-Zip):**
1. Right-click mjr-ehr-system-LIGHTWEIGHT.tar.gz
2. 7-Zip → Extract Here
3. You get: mjr-ehr-interactive/ folder

**Mac:**
```bash
tar -xzf mjr-ehr-system-LIGHTWEIGHT.tar.gz
```

**Linux:**
```bash
tar -xzf mjr-ehr-system-LIGHTWEIGHT.tar.gz
```

### B. Create Installer Folder on USB

**Windows File Explorer:**
1. Open the USB drive (usually E: or F:)
2. Create new folder: `mjr-ehr-installer`

**Mac Finder:**
1. Open USB drive in Finder
2. Create new folder: `mjr-ehr-installer`

**Linux:**
```bash
cd /media/$USER/UBUNTU_DEMO
mkdir mjr-ehr-installer
```

### C. Copy Files to USB

Navigate to the extracted `mjr-ehr-interactive` folder, then:

**Copy these into USB's mjr-ehr-installer/ folder:**

```
mjr-ehr-interactive/mjr-ehr-usb-package/
├── mjr-ehr-complete.tar.gz       ← Copy this
├── setup.sh                       ← Copy this
└── INSTALL_INSTRUCTIONS.txt       ← Copy this
```

**Final USB structure:**
```
USB_DRIVE/
├── (Ubuntu bootable files - don't touch)
└── mjr-ehr-installer/
    ├── mjr-ehr-complete.tar.gz
    ├── setup.sh
    └── INSTALL_INSTRUCTIONS.txt
```

### D. Safely Eject USB

**Windows:** Right-click USB → Eject
**Mac:** Drag USB icon to trash
**Linux:** `umount /media/$USER/UBUNTU_DEMO`

## 🖥️ STEP 4: INSTALL ON TARGET LAPTOP

### A. Boot from USB

1. **Insert USB** into target laptop

2. **Restart laptop**

3. **Press boot menu key** during startup:
   - Dell: F12
   - HP: F9 or ESC
   - Lenovo: F12 or F8
   - ASUS: ESC
   - Acer: F12
   - Generic: Try F12, F9, F8, ESC, or F2

4. **Select USB drive** from boot menu

5. **Ubuntu live screen appears**

### B. Install Ubuntu

1. **Select:** Try or Install Ubuntu

2. **Click:** Install Ubuntu

3. **Language:** English (or your preference)

4. **Keyboard layout:** Select yours

5. **Updates and other software:**
   - ✅ Normal installation
   - ✅ Download updates while installing
   - ✅ Install third-party software

6. **Installation type:**
   - ⚠️ **Select:** Erase disk and install Ubuntu
   - This will DELETE everything on the laptop!
   - Click "Install Now"
   - Confirm by clicking "Continue"

7. **Time zone:** Select yours

8. **Who are you?**
   - Your name: Demo User
   - Computer name: mjr-ehr-demo
   - Username: demo
   - Password: demo1234
   - ✅ Log in automatically

9. **Wait 15-20 minutes** for Ubuntu installation

10. **Click "Restart Now"** when prompted

11. **Remove USB** when it says "Please remove installation medium"

12. **Press ENTER**

### C. Ubuntu First Boot

1. Laptop boots into fresh Ubuntu desktop

2. **Skip** online accounts

3. **Skip** Livepatch

4. **Skip** or complete Welcome wizard

5. **You now have clean Ubuntu installed!**

3. You should see the `mjr-ehr-installer/` folder

1. Press: **Ctrl + Alt + T**

2. Terminal window opens

### C. Navigate to Installer

```bash

# Verify files are there

# - mjr-ehr-complete.tar.gz
# - setup.sh
```

8. ✅ Configure autostart

## 🔧 TROUBLESHOOTING

### "Cannot boot from USB"

**Fix:**
1. Enter BIOS (F2, Del, or F10 during boot)
2. Disable Secure Boot
3. Enable UEFI Boot
4. Set USB as first boot device
5. Save and exit

### "Installation failed - npm errors"

**Fix:**
- Ensure internet connection active
- Retry: `sudo bash setup.sh`

### "System doesn't auto-start"

**Fix:**
```bash
# Manual start
/opt/mjr-ehr/scripts/start-demo.sh

# Or re-run setup
cd /opt/mjr-ehr/scripts
sudo bash setup.sh
```

### "Speech recognition not working"

**Fix:**
- Check microphone connected
- Click microphone icon in Chromium address bar
- Allow microphone access
- Speak clearly into microphone
