## OPTION 2: TRANSFER TO USB FOR UBUNTU INSTALLATION

### What You Need

- USB drive (minimum 2GB for just the app, 8GB+ for bootable installer)
- Windows, Mac, or Linux computer
- Ubuntu laptop to install on (target system)

### Copy to USB Drive

**Windows:**
1. Insert USB drive
2. Open File Explorer
3. Navigate to your Downloads folder
4. Copy `mjr-ehr-system-complete.tar.gz` to USB drive
5. Safely eject USB

**Mac:**
1. Insert USB drive
2. Open Finder
3. Drag `mjr-ehr-system-complete.tar.gz` to USB drive
4. Eject USB

**Linux:**
```bash
# Find USB device
lsblk

# Mount if needed (usually auto-mounts to /media/username/USB_NAME)

# Copy file
cp mjr-ehr-system-complete.tar.gz /media/$USER/YOUR_USB_NAME/

# Sync and unmount
sync
umount /media/$USER/YOUR_USB_NAME
```

## OPTION 3: INSTALL ON UBUNTU SYSTEM

### Prerequisites

- Ubuntu 22.04 LTS or 24.04 LTS installed
- Internet connection (for downloading Node.js)
- Sudo access

### Installation Steps

1. **Insert USB drive into Ubuntu computer**

2. **Open Terminal** (Ctrl+Alt+T)

3. **Navigate to USB:**
   ```bash
   cd /media/$USER/YOUR_USB_NAME
   ls -lh
   # You should see mjr-ehr-system-complete.tar.gz
   ```

4. **Extract the archive:**
   ```bash
   tar -xzf mjr-ehr-system-complete.tar.gz
   cd mjr-ehr-interactive
   ```

5. **Navigate to the USB package:**
   ```bash
   cd mjr-ehr-usb-package
   ls -lh
   # You should see:
   # - mjr-ehr-complete.tar.gz
   # - setup.sh
   # - INSTALL_INSTRUCTIONS.txt
   ```

6. **Run the installer:**
   ```bash
   sudo bash setup.sh
   ```

7. **Wait for installation** (~10-15 minutes):
   - Installs Node.js 20
   - Installs Chromium browser
   - Installs SQLite
   - Extracts MJR-EHR application to /opt/mjr-ehr
   - Installs dependencies
   - Configures autostart
   - Creates desktop launcher

8. **Reboot:**
   ```bash
   sudo reboot
   ```

9. **System auto-starts MJR-EHR in fullscreen!**

## OPTION 4: CREATE BOOTABLE USB (COMPLETE DEMO SYSTEM)

### What You Need

- 8GB+ USB drive
- Ubuntu 24.04 Desktop ISO: https://ubuntu.com/download/desktop
- Rufus (Windows) or Etcher (Mac/Linux)

### Step-by-Step

**A. Download Ubuntu ISO**

1. Visit https://ubuntu.com/download/desktop
2. Download Ubuntu 24.04 LTS Desktop (3GB ISO file)
3. Save to your computer

**B. Create Bootable USB**

**Windows (using Rufus):**
1. Download Rufus: https://rufus.ie
2. Insert 8GB+ USB drive
3. Open Rufus
4. Device: Select your USB drive
5. Boot selection: Click SELECT → choose ubuntu-24.04-desktop-amd64.iso
6. Partition scheme: GPT
7. Target system: UEFI
8. Click START
9. Wait for completion (~10 minutes)

**Mac/Linux (using Etcher):**
1. Download Etcher: https://www.balena.io/etcher
2. Insert USB drive
3. Open Etcher
4. Flash from file → Select ubuntu-24.04-desktop-amd64.iso
5. Select target → Choose USB drive
6. Flash!
7. Wait for completion (~10 minutes)

**C. Add MJR-EHR to Bootable USB**

After creating bootable Ubuntu USB, it will remount:

1. **Create installer folder on USB:**
   ```
   USB_DRIVE/
   └── mjr-ehr-installer/
       └── (we'll put files here)
   ```

2. **Extract your downloaded tar.gz:**
   - Extract `mjr-ehr-system-complete.tar.gz`
   - Navigate into `mjr-ehr-interactive/mjr-ehr-usb-package/`

3. **Copy these 3 files to USB:**
   ```
   USB_DRIVE/mjr-ehr-installer/
   ├── mjr-ehr-complete.tar.gz
   ├── setup.sh
   └── INSTALL_INSTRUCTIONS.txt
   ```

4. **Eject USB safely**

**D. Install on Target Laptop**

1. **Boot from USB:**
   - Insert USB into target laptop
   - Restart computer
   - Press F12, F9, or ESC during boot (depends on manufacturer)
   - Select USB drive from boot menu

2. **Install Ubuntu:**
   - Click "Install Ubuntu"
   - Language: English
   - Keyboard: Your layout
   - Installation type: "Erase disk and install Ubuntu"
   - Create user:
     - Name: demo
     - Password: demo1234
     - Computer name: mjr-ehr-demo
   - Wait for installation (~15-20 minutes)
   - Reboot when prompted

3. **Re-insert USB drive**

4. **Open Terminal** (Ctrl+Alt+T)

5. **Navigate to installer:**
   ```bash
   cd /media/$USER/*/mjr-ehr-installer
   ```

6. **Run installation:**
   ```bash
   sudo bash setup.sh
   ```

7. **Enter password:** demo1234

8. **Wait for installation** (~10 minutes)

9. **Reboot:**
   ```bash
   sudo reboot
   ```

10. **Done!** System boots directly into MJR-EHR fullscreen

## TROUBLESHOOTING

### "Cannot extract .tar.gz file on Windows"

**Solution:** Install 7-Zip
- Download: https://www.7-zip.org/
- Install
- Right-click file → 7-Zip → Extract Here

### "USB not recognized"

**Solution:** 
- Try different USB port
- Reformat USB to FAT32 or exFAT
- Use USB 2.0 port instead of 3.0

### "Permission denied when running setup.sh"

**Solution:**
```bash
chmod +x setup.sh
sudo bash setup.sh
```

### "Cannot boot from USB"

**Solution:**
- Disable Secure Boot in BIOS
- Enable Legacy Boot or UEFI Boot
- Try different USB port
- Recreate bootable USB

### "System doesn't auto-start after installation"

**Solution:**
```bash
# Manual start
/opt/mjr-ehr/scripts/start-demo.sh

# Or re-run setup to fix autostart
sudo bash /opt/mjr-ehr/scripts/setup.sh
```
