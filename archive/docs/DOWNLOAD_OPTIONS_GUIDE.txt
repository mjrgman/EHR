## 🚀 INSTALLATION COMPARISON

### LIGHTWEIGHT Installation:

```bash
# Extract
tar -xzf mjr-ehr-system-LIGHTWEIGHT.tar.gz

# Navigate to installer
cd mjr-ehr-interactive/mjr-ehr-usb-package

# Run installer (downloads dependencies)
sudo bash setup.sh

# Installation downloads ~100MB from npm
# Total time: ~15 minutes

# Reboot
sudo reboot
```

**What happens:**
1. Installs Node.js, Chromium, SQLite
2. Extracts application to /opt/mjr-ehr
3. Downloads node_modules from npm (~100MB)
4. Builds application
5. Configures autostart

### COMPLETE Installation:

```bash

cd mjr-ehr-interactive/mjr-ehr-usb-package

# Run installer (uses included dependencies)
sudo bash setup.sh

# No downloads needed!
# Total time: ~10 minutes

# Reboot
sudo reboot
```

**What happens:**
1. Installs Node.js, Chromium, SQLite
2. Extracts application with node_modules to /opt/mjr-ehr
3. Configures autostart
4. Done!

## ❓ TROUBLESHOOTING

### "Download shows 26MB but should be 27MB"

**This is normal!**
- Browsers show compressed size
- File is actually 27MB when saved
- Check after download: `ls -lh filename.tar.gz`

### "Cannot extract on Windows"

**Solution:**
1. Install 7-Zip: https://www.7-zip.org/
2. Right-click file → 7-Zip → Extract Here
3. Wait for extraction (creates large folder)

### "npm install fails during setup"

**For LIGHTWEIGHT version:**
- Ensure internet connection active
- Retry: `sudo bash setup.sh`

**Alternative:**
- Download COMPLETE version instead
- No internet needed for installation

### "File too large for USB"

**Solutions:**
1. Use LIGHTWEIGHT version (27MB)
2. Use larger USB drive (4GB+)
3. Extract on target computer directly
